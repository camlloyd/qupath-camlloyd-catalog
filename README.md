# QuPath-camlloyd catalog

A catalog of QuPath extensions maintained by Cameron Lloyd.

## Adding this catalog to QuPath

Requires **QuPath 0.7.0**.

1. Launch QuPath
2. Choose *Extensions > Manage extensions*
3. Click *Manage extension catalogs*
4. Click *Add* and enter this URL:
   ```
   https://github.com/camlloyd/qupath-camlloyd-catalog
   ```
